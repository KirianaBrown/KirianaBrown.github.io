BLACKJACK

The BlackJack Project was the first studio project completed in a team based setting.Client Requirements were for an online interactive blackjack game which kept score.

My responsibilities working on this project was the overall design and experience of the application

Technologies used for this project: HTML5, CSS3, JS

User Interface Design

The key to the design was based on a clean/simple design with links to casino felt tables. The button colours were selected to draw attention of the player. Future improvements would be to make this more responsive and fit to screen height to prevent scroll, clean up the add player name section and relocate the score sheet.
