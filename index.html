METHODOLOGY

Lets start off with having a conversation followed by doing some user and stakeholder research to identify pain points and what is on the wish list for the design and functionality of the product. Once we have collated this lets jump into the backlog and pulling out user stories, creating personas, wireframes and prototypes. Lets next aim to get a minimal viable product delivered to you which can be user tested and built upon - adding new features and designs as we cycle through an agile cycle of flexibility.
