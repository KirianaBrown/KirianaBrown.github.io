Cerebral Fix

Studio Project - Location Device Web App

Cerebral Fix is a Christchurch Game, VR and App Development company. To test their mobile apps they have a range of devices, shared between various development and testing teams. The existing whiteboard-based system for tracking their devices was not effective, especially with the company now split between two offices.

We used Flask (a Python micro-framework) to develop a web app which was hosted on Heroku and connected to a PostgreSQL database. The app required a valid login before allowing users to search the list of devices and filter by brand, form factor, spec quality and OS version. A user could then identify who had the device and/or assign the device to themselves. Our app had comprehensive management tools for users and devices, including add, remove, history and default options.

My role within the project team was as design and layout developer creating the front end styled aspects of the UI.

  • Date: April 2019
  • Client: Cerebral Fix
  • Category: Studio Project
